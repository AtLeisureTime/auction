Auction rules

Admins create lots and plans auction time for every lot independently. Lots with "Draft" auction state are invisible for users.
Users can make stakes on lot when lot auction is in "Started" state. Use filters on "Lots" page to view auctions.
If there are any not processed stakes of other users in the queue at the current moment, the button "Minimal Stake Exceeding Any Other Stake" make minimal stake exceeding the highest one in the queue.
Winner of "Finished" auction is user with the highest stake.
